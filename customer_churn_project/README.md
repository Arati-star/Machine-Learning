# 📊 Customer Churn Prediction

## 📌 Project Overview

This project predicts whether a telecom customer is likely to churn using Machine Learning.

The application is built using Python, Scikit-Learn, and Streamlit. Users can enter customer information through a web interface and receive the churn prediction along with the probability.

---

## 🚀 Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Pipeline
- OneHotEncoder
- Random Forest Classifier
- Hyperparameter Tuning
- ROC-AUC Evaluation
- Streamlit Web Application
- Business Recommendation
